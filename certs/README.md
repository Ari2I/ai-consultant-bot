# Сертификат для GigaChat API

Положите сюда файл корневого сертификата НУЦ Минцифры (`.pem` или
`.cer`), скачанный с https://www.gosuslugi.ru/crt, и укажите путь к
нему в `.env`:

```
GIGACHAT_CA_BUNDLE_FILE=/app/certs/russian_trusted_root_ca.cer
```

(путь `/app/certs/...` — для запуска в Docker, см.
`docker-compose.yml`; при локальном запуске без Docker укажите
реальный путь на вашей машине).

Сам файл сертификата в репозиторий не включён и исключён из git
(`.gitignore`) — скачайте актуальную версию с официального источника
самостоятельно. Подробности — в `DEPLOYMENT.md`, раздел «Сертификат
для GigaChat».
